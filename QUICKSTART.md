# 🚀 快速启动指南

## 第一步：打开项目

1. 打开**微信开发者工具**
2. 点击**导入项目**
3. 选择目录：`/Users/len/Projects/mama-praise-me`
4. AppID：选择**测试号**或输入你的AppID
5. 点击**导入**

## 第二步：配置云开发

1. 点击工具栏的**云开发**按钮
2. 如果提示开通，按提示开通云开发
3. 确认云环境ID（已配置：cloud1-7gebc7vu6f38e22a）

## 第三步：创建数据库集合

在云开发控制台 → 数据库，创建以下集合（点击"添加集合"）：

1. **children** - 孩子信息
2. **tasks** - 任务定义
3. **task_completions** - 任务完成记录
4. **prizes** - 奖品定义
5. **redemptions** - 兑换记录
6. **coin_records** - 金币记录

> 权限选择：仅创建者可读写

## 第四步：部署云函数

在微信开发者工具中，依次部署以下云函数：

1. **login**
   - 右键 `cloudfunctions/login`
   - 选择"上传并部署：云端安装依赖"

2. **manageTasks**
   - 右键 `cloudfunctions/manageTasks`
   - 选择"上传并部署：云端安装依赖"

3. **manageCoins**
   - 右键 `cloudfunctions/manageCoins`
   - 选择"上传并部署：云端安装依赖"

4. **managePrizes**
   - 右键 `cloudfunctions/managePrizes`
   - 选择"上传并部署：云端安装依赖"

5. **manageRedemptions**
   - 右键 `cloudfunctions/manageRedemptions`
   - 选择"上传并部署：云端安装依赖"

6. **manageChildren**
   - 右键 `cloudfunctions/manageChildren`
   - 选择"上传并部署：云端安装依赖"

> 部署每个云函数大约需要1-2分钟，请耐心等待

## 第五步：运行项目

1. 点击工具栏的**编译**按钮
2. 查看模拟器中的小程序
3. 开始测试！

## 测试流程

### 1. 添加孩子
1. 点击底部的**设置**Tab
2. 点击**孩子管理**
3. 点击右下角的**+**按钮
4. 输入孩子信息（姓名、头像、年龄）
5. 点击**保存**

### 2. 创建任务
1. 返回首页，点击右上角的**📝**按钮
2. 点击**添加任务**
3. 填写任务信息：
   - 任务名称：每天阅读30分钟
   - 任务描述：阅读任何书籍都可以
   - 金币奖励：10
   - 任务类型：每日任务
4. 点击**保存**

### 3. 完成任务
1. 在首页任务列表中找到刚才创建的任务
2. 点击任务卡片
3. 确认完成
4. 查看金币增加！

### 4. 创建奖品
1. 点击底部的**奖品**Tab
2. 点击**添加奖品**
3. 填写奖品信息：
   - 奖品名称：乐高积木
   - 奖品描述：大型乐高积木套装
   - 所需金币：100
   - 奖品分类：玩具
4. 点击**保存**

### 5. 兑换奖品
1. 在奖品商城找到刚才创建的奖品
2. 点击**兑换**按钮
3. 确认兑换
4. 查看金币扣除！

## ❓ 常见问题

### Q: 云函数部署失败？
A:
- 检查网络连接
- 确认云开发已开通
- 尝试重新部署

### Q: 数据库集合创建失败？
A:
- 确认云开发已开通
- 检查集合名称是否正确
- 尝试刷新页面

### Q: 页面显示空白？
A:
- 检查云函数是否部署成功
- 检查数据库集合是否创建
- 查看控制台错误信息

### Q: 功能无法使用？
A:
- 确认已登录（点击设置 → 登录）
- 检查云函数调用是否成功
- 查看云函数日志
