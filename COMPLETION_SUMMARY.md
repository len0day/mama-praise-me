## 📱 使用流程

### 1. 初始设置
1. 打开小程序
2. 点击"设置" → "孩子管理"
3. 添加孩子（姓名、头像、年龄）
4. 选择当前孩子

### 2. 创建任务
1. 点击首页右上角"📝"按钮
2. 进入任务管理页
3. 点击右下角"+"按钮
4. 填写任务信息：
   - 任务名称（必填）
   - 任务描述（可选）
   - 金币奖励
   - 任务类型（每日/每周/每月）
   - 时间范围（每周/每月任务需要）
5. 点击"保存"

### 3. 完成任务
1. 在首页查看今日任务
2. 点击任务卡片
3. 确认完成
4. 查看金币增加动画

### 4. 创建奖品
1. 点击底部"奖品"Tab
2. 点击右上角"+"按钮
3. 填写奖品信息：
   - 奖品名称（必填）
   - 奖品描述（可选）
   - 奖品图片（可选）
   - 所需金币
   - 奖品分类
   - 库存数量
4. 点击"保存"

### 5. 兑换奖品
1. 在奖品商城浏览奖品
2. 查看金币是否足够
3. 点击"兑换"按钮
4. 确认兑换
5. 查看兑换记录

## 🚀 部署清单

### 在微信开发者工具中完成以下步骤：

#### 1. 创建数据库集合
在云开发控制台 → 数据库，创建6个集合：
- children（孩子信息）
- tasks（任务定义）
- task_completions（任务完成记录）
- prizes（奖品定义）
- redemptions（兑换记录）
- coin_records（金币记录）

权限选择：**仅创建者可读写**

#### 2. 部署云函数
依次部署6个云函数（右键 → 上传并部署：云端安装依赖）：
- login
- manageTasks
- manageCoins
- managePrizes
- manageRedemptions
- manageChildren

#### 3. 配置云环境
确认 `app.js` 中的云环境ID为：`cloud1-4g50c2bp6d01c14d`

#### 4. 编译运行
点击"编译"按钮，开始测试！
